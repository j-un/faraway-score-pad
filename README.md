# faraway-score-pad

ボードゲーム「ファラウェイ」のスコアパッドをWebアプリケーション化したものです。<br>
付属のスコアパッド用紙を利用せずにゲームをプレイできます。<br>
ファラウェイについて: https://www.engames-s.com/product/2968

This is a web application version of the score pad for the board game "Faraway".<br>
You can play the game without using the included paper score pad.<br>
About Faraway: https://site-export.blackrockgames.fr/game/CAT079FA
